# Hook Swipe File Template

## Hook Types

### 1. Result Hooks
- "How I [achieved result] in [timeframe] without [common sacrifice]"
- "I went from [before] to [after] in [timeframe] using [mechanism]"
- "The exact [method] I used to [result]"

### 2. Curiosity Hooks
- "What [authority] doesn't want you to know about [topic]"
- "The [adjective] truth about [topic]"
- "Why most [target audience] will never [desired outcome]"

### 3. Contrast Hooks
- "I was [negative state] until I discovered [mechanism]"
- "The difference between [common approach] and [better approach]"
- "Stop doing [common mistake]. Start doing [better action]."

### 4. Story Hooks
- "In [year], I was [situation]. Then [event] changed everything."
- "The [number] [things] that [result]"
- "When [person] told me [statement], I didn't believe them. Then..."

### 5. Direct Hooks
- "Want [desired outcome]? Read this."
- "If you're [target audience], this is for you."
- "Stop scrolling if you want [benefit]."
